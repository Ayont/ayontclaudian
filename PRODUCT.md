# Product

## Register

product

## Users

Obsidian power users, developers, researchers, and creators who work with several coding-agent CLIs inside a long-lived vault. They need to understand which provider is active, what an agent is doing, what changed, and whether a multi-agent mission is healthy without leaving their work.

## Product Purpose

Claudian turns an Obsidian vault into a provider-neutral agent workspace. It unifies chat, file operations, model switching, autonomous workflows, multi-agent missions, memory, and diagnostics while keeping provider provenance and system state visible. Success means users can act quickly and confidently without wondering which model is running or whether the interface registered their action.

## Brand Personality

Calm, capable, precise. The product should feel like a native professional instrument with distinct provider identities, not a themed chatbot or a decorative AI dashboard.

## Anti-references

- Generic SaaS dashboards made from identical floating cards.
- Neon cyberpunk agent consoles, excessive glow, glass, gradients, or decorative animation.
- Interfaces that hide provider provenance or rely on color alone.
- Dense developer tools with weak hierarchy, tiny hit targets, and silent state changes.
- Provider branding that overwhelms Obsidian or makes different screens feel like different products.

## Design Principles

1. State is a first-class surface: every important action gets immediate, persistent, and understandable feedback.
2. Provider identity is provenance: show it consistently with name, mark, and restrained color.
3. One operating system, many agents: provider personalities vary inside one stable component language.
4. Progressive depth: keep the default view calm, then reveal live details, metrics, and diagnostics on demand.
5. Vault-native confidence: respect Obsidian themes, keyboard behavior, reduced motion, and compact sidebars.

## Accessibility & Inclusion

Target WCAG 2.2 AA. Never encode status only by color. Preserve visible keyboard focus, semantic labels, 44px primary targets where space permits, readable contrast in light and dark themes, and reduced-motion behavior.
